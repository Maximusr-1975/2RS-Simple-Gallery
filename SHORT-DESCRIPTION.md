# Simple Lightbox Gallery

A lightweight, dependency-free JavaScript image gallery with smooth animations and mobile support.

**Features:**
- Pure JavaScript (no jQuery required)
- Only ~8KB total size
- Keyboard navigation (arrows, ESC)
- Touch gestures (swipe)
- Image captions & counter
- Fully responsive
- Easy Bootstrap integration

**Perfect replacement for Fancybox** - faster, lighter, and completely free.

---

## Quick Start

```html
<!-- Add files -->
<link rel="stylesheet" href="simple-lightbox.css">
<script src="simple-lightbox.js"></script>

<!-- Use on images -->
<a href="full-image.jpg" data-lightbox="gallery" data-caption="Description">
    <img src="thumbnail.jpg" alt="Description">
</a>
```

That's it! Auto-initializes on page load.
