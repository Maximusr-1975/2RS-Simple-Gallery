Integration Code for Bootstrap Studio

Step 1: Add files to Bootstrap Studio
  1. In Bootstrap Studio: Design → Custom Code → CSS Files
  2. Import: 2RS_Simple_Gallery.css
  3. In Bootstrap Studio: Design → Custom Code → JavaScript Files
  4. Import: 2RS_Simple_Gallery.js
Step 2: Replace Fancybox with Simple Lightbox

Find all links like:

<a data-fancybox="gallery" href="image.jpg">
  <img src="thumbnail.jpg" alt="Description">
</a>

And replace with:

<a data-lightbox="gallery" href="image.jpg" data-caption="Description">
  <img src="thumbnail.jpg" alt="Description">
</a>
Step 3: Add PDF Support

For PDF documents, use:

<a href="brochure.pdf" 
   data-lightbox="gallery" 
   data-type="pdf"
   data-caption="PDF Title"
   class="btn btn-primary">
    <i class="fas fa-file-pdf"></i> View Brochure
</a>
Step 4: Remove Fancybox

Delete these lines from HTML:

<!-- REMOVE: -->
<link rel="stylesheet" href="https://cdn.jsdelivr.net/.../fancybox.css">
<script src="https://cdn.jsdelivr.net/.../fancybox.umd.js"></script>

<!-- Also remove JavaScript initialization code: -->
<script>
  Fancybox.show([...]);
</script>
