W karuzeli Bootstrap (zastępuje Fancybox)

Kod do wklejenia w Bootstrap Studio

Krok 1: Dodaj pliki w Bootstrap Studio
  1. W Bootstrap Studio: Design → Custom Code → CSS Files
  2. Importuj: simple-lightbox.css
  3. W Bootstrap Studio: Design → Custom Code → JavaScript Files
  4. Importuj: simple-lightbox.js
Krok 2: Zamień Fancybox na Simple Lightbox

Znajdź wszystkie linki typu:

<a data-fancybox="gallery" href="obrazek.jpg">
  <img src="miniaturka.jpg" alt="Opis">
</a>

I zamień na:

<a data-lightbox="gallery" href="obrazek.jpg" data-caption="Opis">
  <img src="miniaturka.jpg" alt="Opis">
</a>
Krok 3: Usuń Fancybox

Usuń następujące linie z HTML:

<!-- USUŃ: -->
<link rel="stylesheet" href="https://cdn.jsdelivr.net/.../fancybox.css">
<script src="https://cdn.jsdelivr.net/.../fancybox.umd.js"></script>
Opcjonalne dostosowanie kolorów do 2R Systems

Dodaj do custom CSS w Bootstrap Studio:

/* Dostosowanie kolorów do marki 2R Systems */
.simple-lightbox-close,
.simple-lightbox-prev,
.simple-lightbox-next {
    background: rgba(0, 102, 204, 0.2);
    border-color: rgba(0, 102, 204, 0.5);
}

.simple-lightbox-close:hover,
.simple-lightbox-prev:hover,
.simple-lightbox-next:hover {
    background: rgba(0, 102, 204, 0.4);
    border-color: #0066cc;
}

.simple-lightbox-caption {
    background: rgba(0, 102, 204, 0.8);
}

.simple-lightbox-counter {
    background: rgba(0, 102, 204, 0.8);
}
